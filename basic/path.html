절대 경로 : 리소스(Resource)를 찾는 파일 위치에 상관없이 항상 같은 위치에 있는 리소스를 찾을 수 있게 해주는 경로 : 기본 주소(도메인)부터 시작
Ex) https://www.youtube.com/embed/9EsKLveY3Rc

상대 경로 : 리소스(Resource)를 찾는 파일의 위치를 기준으로 찾게 해주는 경로
    - ../ : 한 단계 상위 폴더, Ex) 현재 images 폴더, ../images/picture.jpg
    - ./ 현재 폴더, Ex) 현재 images 폴더, ./picture.jpg
    
    - Root 상대경로 : / (Local - C:)
    
IP(Internet Protocol) 주소
    - 실제 인터넷 주소
    - .으로 구분된 4개 부분의 숫자 구성 : 192.168.0.1
    - 각 부분의 숫자는 0-255의 숫자로 구성됨
    - 버전 : IPv4, IPv6

도메인(Domain)
    - 의미있는 단어를 가지고 주소를 구성
    - 실제 주소는 아님
    - 도메인 -> IP : DNS(Domain Name Service)
    - 3구역 : www.naver.com
    - 4구역 : www.lg.co.kr
    - 도메인 대행 업체
        - https://hosting.kr/
        - https://domain.whois.co.kr/
        - https://domain.gabia.com/
