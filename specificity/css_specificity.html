우선순위
element : 1
.class  : 10
#id     : 100
inline  : 1000
*       : 0

Ex)
<div class="box" id="id-box" style="color: yellow;">box</div>   : 1000
.box{color:red;}        : 10
#id-box(color:blue;)    : 100
div{color:green;}       : 1
=>
color : yellow 적용

#id-box.box : 100 + 10 = 110
div#id-box.box : 100 + 10 + 1 = 111
#content h1 : 100 + 1 = 101

ul > li > a : 1 + 1 + 1 = 3 
    
cascading 규칙
 - 코딩 입력, 실행 순서 : 위 => 아래
 - 같은 대상에 대해서 css styling 여러번 적용될 때 가장 마지막에 적용된 style이 적용됨
    
상속
 - 부모 요소에 적용된 style 속성들이 자식 요소에 똑같이 적용되는 것
 - 모든 스타일이 모든 HTML 요소에 상속되는 것은 아님
